## Zoom Tiles

## Supported Actions and Feedbacks

| Category | Name | Description |
|----------|------|-------------|
| Actions | Favorite by Index | Favorite a tile using the gallery index and tile index. |
| Actions | Favorite by User Name | Favorite user(s) by User Name. For multiple users, separate names with a comma. |
| Actions | Replace by User Name | Replace user in the gallery with a new user regardless of where they are in the gallery. |
| Actions | Start Personal Meeting ID | Start a meeting using your Personal Meeting ID |
