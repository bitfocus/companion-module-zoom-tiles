## Zoom Tiles

## Supported Actions and Feedbacks

| Category | Name | Description |
|----------|------|-------------|
| action-capture | Start Capture Engine | Start the capture engine to begin capturing video tiles. |
| action-capture | Stop Capture Engine | Stop the capture engine to end capturing video tiles. |
| action-config | Load Configuration | Load a configuration file from an absolute path on disk. |
| action-favorite | Favorite by Index | Favorite a tile using the gallery index and tile index. |
| action-favorite | Favorite by User Name | Favorite user(s) by User Name. For multiple users, separate names with a comma. |
| action-gallery-timer | Activate Hole Punch | Activate hole punch for a specific gallery view. |
| action-gallery-timer | Deactivate Hole Punch | Deactivate Hole Punch for a specific gallery view. |
| action-gallery-timer | Disable Gallery | Disable a specific gallery view. |
| action-gallery-timer | Enable Gallery | Enable a specific gallery view. |
| action-gallery-timer | Start Queue Timer | Start the queue timer for gallery updates. |
| action-gallery-timer | Stop Queue Timer | Stop the queue timer for gallery updates. |
| action-join-leave | Join Meeting | Join a meeting with Meeting ID, Password, and Display Name |
| action-join-leave | Leave Meeting | Leave the current meeting |
| action-join-leave | Start Instant Meeting | Start an instant meeting |
| action-join-leave | Start Personal Meeting ID | Start a meeting using your Personal Meeting ID |
| action-join-leave | ZAK Join Meeting | Join a meeting using ZAK token, Meeting ID, Password, and Display Name |
| action-replace | Replace by User Name | Replace user in the gallery with a new user regardless of where they are in the gallery. |
| action-replace | Replace Tile Index with Tile Index | Replace a Tile Index with another Tile Index. |
| action-replace | Replace User with Tile Index | Replace a user with a specific gallery/tile index. |
| Actions | Block by Index | Block a tile using the gallery index and tile index. |
| Actions | Block by User Name | Block user(s) by User Name. For multiple users, separate names with a comma. |
